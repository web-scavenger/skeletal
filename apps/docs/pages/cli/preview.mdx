# `skeletal-ui preview`

Starts a local server that renders all generated skeletons side-by-side for visual review.

```sh
npx skeletal-ui preview
```

## When to use

Use to visually inspect all generated skeletons before committing. Useful for:

- Checking that skeleton sizes match the real layout
- Comparing multiple skeletons at once
- Reviewing after a bulk `analyze` run

## Usage

```sh
npx skeletal-ui preview
# → Server running at http://localhost:4001
```

Open the URL in your browser to see all skeletons rendered in a grid.
