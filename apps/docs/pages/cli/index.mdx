# CLI Reference

| Command | Description |
|---|---|
| [`init`](/cli/init) | Interactive setup, creates `skeletal.config.ts` |
| [`analyze`](/cli/analyze) | Scan, crawl, generate, and wire skeletons |
| [`check`](/cli/check) | Assert all skeletons are up to date |
| [`watch`](/cli/watch) | Re-analyze on source file changes |
| [`preview`](/cli/preview) | Browse all skeletons locally |
| [`eject`](/cli/eject) | Copy a skeleton into source for manual editing |

## Global install

```sh
npm install -g skeletal-ui
skeletal-ui analyze
```

Or use without installing:

```sh
npx skeletal-ui analyze
```
